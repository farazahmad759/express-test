## Method 1

(Automatic exam step-by-step)

- Select `Grade`
- Select `Subject`
- Select `Syllabus`
- Select `Exam Preset` (Exam Presets depend on `Grade`, `Subject` and `Syllabus`)
- Select `Topic(s)`

When an exam has been created, the user can double-click on a question to replace it with new question.
